# VS Code Jupyter Notebook Shortcuts

## Command Mode (press `Esc` first)

| Shortcut  | Action                  |
| --------- | ----------------------- |
| `A`       | Insert cell above       |
| `B`       | Insert cell below       |
| `DD`      | Delete cell             |
| `Z`       | Undo cell delete        |
| `M`       | Change cell to Markdown |
| `Y`       | Change cell to Code     |
| `Enter`   | Enter edit mode         |
| `↑` / `↓` | Navigate between cells  |
| `L`       | Toggle line numbers     |

## Edit Mode (press `Enter` on a cell)

| Shortcut       | Action                  |
| -------------- | ----------------------- |
| `Esc`          | Enter command mode      |
| `Ctrl+Enter`   | Run cell                |
| `Shift+Enter`  | Run cell & select below |
| `Alt+Enter`    | Run cell & insert below |
| `Ctrl+Shift+[` | Collapse cell           |
| `Ctrl+Shift+]` | Expand cell             |
| `Tab`          | Autocomplete / indent   |
| `Shift+Tab`    | Show tooltip / dedent   |
| `Ctrl+/`       | Toggle comment          |
| `Ctrl+Shift+K` | Delete line             |
| `Ctrl+D`       | Select next occurrence  |

## General

| Shortcut             | Action                         |
| -------------------- | ------------------------------ |
| `Ctrl+S`             | Save                           |
| `Ctrl+Shift+P`       | Command Palette                |
| `Ctrl+Shift+V`       | Markdown preview               |
| `0 0` (command mode) | Restart kernel                 |
| `Ctrl+K Ctrl+S`      | Open Keyboard Shortcuts editor |
